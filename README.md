# Automated Email Header Analyzer

Автоматический инструмент на Python для анализа технических заголовков (headers) электронных писем. Скрипт помогает выявлять фишинг, спуфинг (подмену отправителя) и анализировать маршрут следования письма (Hops).


Инструмент анализирует критические маркеры кибербезопасности:
* **Валидация инфраструктурных политик:** Проверка статусов SPF, DKIM и DMARC на основе заголовков `Authentication-Results`.
* **Детект спуфинга (From vs Return-Path):** Сравнение отображаемого домена автора (`From`) с техническим доменом возврата ошибок (`Return-Path`).
* **Анализ цепочки серверов (Received Hops):** Извлечение всех IP-адресов серверов, через которые прошло письмо, для отслеживания истинного источника отправления.
* **Интеллектуальный вердикт:** Выставление оценки безопасности (`БЕЗОПАСНО`, `ПОДОЗРИТЕЛЬНО`, `ОПАСНО`).

## Как запустить

Скрипт написан с использованием только стандартных библиотек Python, поэтому установка дополнительных зависимостей не требуется.

1. Клонируйте репозиторий или скачайте файл `email_analyzer.py`.
2. SЗапустите скрипт через терминал или командную строку:
   ```bash
   python email_analyzer.py
